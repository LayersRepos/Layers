\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_layers_1_1_widget}{Widget}} class wraps a QWidget with a \mbox{\hyperlink{class_layers_1_1_themeable}{Themeable}} to give QWidgets compatibility with the Layers theme system. The Layers \mbox{\hyperlink{class_layers_1_1_widget}{Widget}} class overrides the QWidget\textquotesingle{}s \mbox{\hyperlink{class_layers_1_1_widget_ad9da6e86a017fc03e607effec4a87f16}{paint\+Event()}} and uses the attributes provided by the \mbox{\hyperlink{class_layers_1_1_themeable}{Themeable}} class to handle the widget\textquotesingle{}s appearance.
