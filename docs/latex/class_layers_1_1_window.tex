Applies the given theme to the caller and its children.

This function works recursively to apply the given theme to the caller and children in the caller\textquotesingle{}s hierarchy.

A name must be set with \mbox{\hyperlink{class_layers_1_1_themeable_a78f9b10993fcef1557aeba1bc75e5c90}{set\+\_\+name()}} before themes can be applied. This is because a name is the minimum requirement to construct the theme tag.

Initializes the reference list to child themeables.

A list of child themeable references is necessary to allow the \mbox{\hyperlink{class_layers_1_1_window_a5a0badc67b976fa520d51550429362c2}{apply\+\_\+theme()}} function to work recursively.

Classes that inherit the \mbox{\hyperlink{class_layers_1_1_themeable}{Themeable}} class should define this function and use \mbox{\hyperlink{class_layers_1_1_themeable_ab23083741e63ca400bb9860b83609840}{add\+\_\+child\+\_\+themeable\+\_\+reference()}} to populate the reference list.

This function is called by \mbox{\hyperlink{class_layers_1_1_themeable_afdae3d404debc794625aa2f633ec23bc}{init\+\_\+themeable()}}.
