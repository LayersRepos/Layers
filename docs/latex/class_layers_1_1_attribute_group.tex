\doxysubsection{Detailed Description}
Implements a container used to define a group of Attributes.

The Attributes are stored in a QMap of QString-\/\+Attribute$\ast$ pairs where the QString matches the name of the associated \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}}.

A group of Attributes is stateful if a single \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}} in the group is stateful.
