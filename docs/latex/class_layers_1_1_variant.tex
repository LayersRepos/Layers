\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_layers_1_1_variant}{Variant}} type with signal/slot support

The \mbox{\hyperlink{class_layers_1_1_variant}{Variant}} class is a QObject that wraps a QVariant to enable Qt signal/slot functionality.

The purpose of having a variant type with signal/slot support is to enable value change detection. Any time the QVariant is modified, the changed() signal gets emitted.
