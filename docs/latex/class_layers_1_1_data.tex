\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_layers_1_1_data}{Data}} type which represents either a single \mbox{\hyperlink{class_layers_1_1_variant}{Variant}} or multiple Variants

A \mbox{\hyperlink{class_layers_1_1_data}{Data}} object is a QObject that stores a pointer to either a single \mbox{\hyperlink{class_layers_1_1_variant}{Variant}} or a QVariant\+Map where QString represents the state of the \mbox{\hyperlink{class_layers_1_1_variant}{Variant}}.

\mbox{\hyperlink{class_layers_1_1_data}{Data}} associated with multiple Variants is stateful, whereas \mbox{\hyperlink{class_layers_1_1_data}{Data}} associated with a single \mbox{\hyperlink{class_layers_1_1_variant}{Variant}} is not stateful. This is referred to as the \mbox{\hyperlink{class_layers_1_1_data}{Data}}\textquotesingle{}s statehood or statefulness.
