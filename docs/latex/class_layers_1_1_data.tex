\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_layers_1_1_data}{Data}} type

A \mbox{\hyperlink{class_layers_1_1_data}{Data}} object is a QObject that stores a pointer to either a single \mbox{\hyperlink{class_layers_1_1_variant}{Variant}} or a map of QString-\/\+Variant pairs where the QString represents the state associated with the \mbox{\hyperlink{class_layers_1_1_variant}{Variant}}.

\mbox{\hyperlink{class_layers_1_1_data}{Data}} storing a pointer to a state-\/\+Variant map is stateful, whereas \mbox{\hyperlink{class_layers_1_1_data}{Data}} storing a pointer to a single \mbox{\hyperlink{class_layers_1_1_variant}{Variant}} is not stateful. This can be referred to as the \mbox{\hyperlink{class_layers_1_1_data}{Data}}\textquotesingle{}s statefulness.

The term \textquotesingle{}value\textquotesingle{} is sometimes used synonymously with \mbox{\hyperlink{class_layers_1_1_variant}{Variant}}, although, technically the value is stored in a QVariant stored within the \mbox{\hyperlink{class_layers_1_1_variant}{Variant}}. For example, the \mbox{\hyperlink{class_layers_1_1_data_a09151718594ab891937cdbb722c8f259}{set\+\_\+value()}} function takes a QVariant argument intended to represent the value being set.
