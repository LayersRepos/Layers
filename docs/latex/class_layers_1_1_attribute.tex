\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}} implementation used throughout Layers by \mbox{\hyperlink{class_layers_1_1_attribute_group}{Attribute\+Group}}, \mbox{\hyperlink{class_layers_1_1_theme}{Theme}}, and \mbox{\hyperlink{class_layers_1_1_themeable}{Themeable}}.

An \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}} stores a pointer to a \mbox{\hyperlink{class_layers_1_1_data}{Data}} object which is used for drawing Themeables. At first, an \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}} inializes \mbox{\hyperlink{class_layers_1_1_data}{Data}} and {\itshape owns} it. However, through \mbox{\hyperlink{class_layers_1_1_attribute_a71009df30061fc4a7a68bd22be06d3d1}{get\+\_\+data\+\_\+from()}}, an \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}} can be made to point to \mbox{\hyperlink{class_layers_1_1_data}{Data}} from another \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}}.
