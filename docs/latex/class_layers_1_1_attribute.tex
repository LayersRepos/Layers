\doxysubsection{Detailed Description}
Provides structure for attributes to be used by Themes and Themeables.

Attributes store pointers to Variants which store values used for painting Themeables and supporting their appearance.

A \mbox{\hyperlink{class_layers_1_1_themeable}{Themeable}} stores a set of \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}} pointers, and a \mbox{\hyperlink{class_layers_1_1_theme}{Theme}} stores a collection of \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}} pointer sets for many Themeables.

An \mbox{\hyperlink{class_layers_1_1_attribute}{Attribute}} can either point to a single \mbox{\hyperlink{class_layers_1_1_variant}{Variant}} or point to a map of QString-\/\+Variant pairs where the QStrings are states.
