\doxysubsection{Detailed Description}
An \mbox{\hyperlink{class_layers_1_1_entity}{Entity}} is a QObject that is a named, disableable, and stateable.

Named means that an \mbox{\hyperlink{class_layers_1_1_entity}{Entity}} has a name.

Disableable means that an \mbox{\hyperlink{class_layers_1_1_entity}{Entity}} can be switched between on/off states.

Stateable means that an \mbox{\hyperlink{class_layers_1_1_entity}{Entity}} can be in different states.
