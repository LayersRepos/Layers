Detailed Description

Provides structure for attributes to be used by Themes and Themeables.

Attributes store pointers to Variants which store values used for painting Themeables and supporting their appearance.

A Themeable stores a set of Attribute pointers, and a Theme stores a collection of Attribute pointer sets for many Themeables.

An Attribute can either point to a single Variant or point to a map of QString-Variant pairs where the QStrings are states.
